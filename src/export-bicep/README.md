# Bicep Export

This folder contains all the necessary files for implementing the Azure infrastructure using Bicep.

## Contents

- **prompts/** - Ready-to-use prompts for coding agents
- **instructions/** - Step-by-step implementation guidance
- **decisions/** - Architecture Decision Records (ADRs)
- **configurations/** - Component configurations and parameters
- **deployment/** - CI/CD pipeline configurations
- **context/** - Product overview and context
